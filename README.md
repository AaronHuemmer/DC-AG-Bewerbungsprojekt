# TSC Pottenstein Mini-Shop

## Setup
1. Datenbank anlegen:
   - phpMyAdmin öffnen
   - neue DB `shop` erstellen
   - `shop.sql` importieren

2. Shop starten:
   ```bash
   php -S localhost:8000 -t www
   ```

3. Im Browser öffnen: [http://localhost:8000](http://localhost:8000)

## Funktionen
- Produkte anzeigen
- Warenkorb mit jQuery
- Bestellung speichern
